email_spy

A simple gem that allows you to catch emails sent from your Rails app into your browser.

Based entirely on letter_opener, with pull requests, bug fixes, and real versioning.

Installation

Add this line to your application's Gemfile:

gem 'email_spy'

And then execute:

$ bundle

Or install it yourself as:

$ gem install email_spy

Usage

To start using email_spy simply add this line to your config/environments/development.rb file:

  config.action_mailer.delivery_method = :email_spy

That's it! Whenever your development mode application sends an email through action_mailer it'll open the default browser.
